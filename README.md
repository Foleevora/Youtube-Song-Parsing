# Youtube-Song-Parsing

파이썬 라이브러리 Beautifulsoup를 공부하기 위한 프로그램입니다.

# Environment

* Mac OS
* Python 3.6.1

# Setup
    $ pip install beautifulsoup4

# Usage
    $ python3 YoutubeSongInfo.py [Url]

# Example
    $ python3 YoutubeSongInfo.py "https://www.youtube.com/watch?v=obzb7nlpXZ0"
